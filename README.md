# Introduction

## Template Usage

1. Go to [gitbook-template](https://github.com/airdb-template/gitbook)
2. Click `Use this template`
3. Configure `GithubBotToken` for github action. `Setting` -> `Secrets` -> `Add
   a new secret`


## Gitbook

### Installation

```
sudo npm install -g gitbook-cli
gitbook install
```

### Run and Build

```
gitbook serve
gitbook build
```
